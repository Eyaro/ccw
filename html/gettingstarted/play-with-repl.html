Play with a REPL

Launch the REPL

  • Select your project's node
  • In the context menu: Run as ... > Run configurations > Clojure > New (in Clojure node's contextual menu)
  • Click run

A console should open, with Clojure's prompt waiting for you ...

Use the Namespace Browser

The namespace browser will show you all symbols of all namespaces of the REPL you have launched, and allow you to jump to the source code defining them, if available (double-click on the symbol).

It also allows you to search the symbol you want by typing regular expressions in the Find text field, and choosing to do the search only on symbols names, or also on symbols documentation.

If you let the mouse hover over a node in the browser, you will have the documentation, if available.

  • To see the namespace browser, you must
    • either select it via menu Window > Show View > Other > Clojuredev Views > Namespace Browser
    • either open the java perspective. The namespace browser should be located behind the outline view by default (if you are in the java perspective and don't see the namespace browser, you may reset the java perspective to its default value via menu Window > Reset Perspective, or find it manually via menu Window > Show View > Namespace Browser

How does it work?

When you have launched the REPL, the Clojure Eclipse plugin has embedded "server code" in the launched Clojure environment. This server code is contacted by the Clojure Eclipse plugin to give information on the running Clojure environment. The namespace browser uses this server to provide you with the most possible up to date info on the symbols.

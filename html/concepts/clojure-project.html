Clojure Projects

A Clojure project is just a Java project, with an additional builder for Clojure code.

Creating a Clojure project or enabling Clojure support for an existing Java project will check if you have the Clojure JAR files in your classpath, and if not, it will add a dependency on the Clojure plugin's own embedded versions of Clojure and the user-contributed Clojure libraries (a.k.a. "clojure-contrib").
